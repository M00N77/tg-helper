Как работает бот в групповом чате
====================================

1. Первичная настройка
-----------------------
Кто может: любой участник чата

  /i_am_director

Бот создаёт команду, назначает вызвавшего директором и привязывает чат.
После этого все участники чата автоматически становятся членами команды
при первом сообщении.

Навигация по группам: если бот работает в нескольких группах, каждая
группа — отдельная команда со своей доской YouGile и настройками.


2. Подключение канбан-доски (YouGile)
---------------------------------------
Кто может: только директор (админ)

Доступно три способа, от самого безопасного до самого быстрого:

Способ А (рекомендуемый) — через личный чат с ботом:
  1. В группе: /setup_yougile
  2. Бот присылает inline-кнопку со ссылкой на ЛС
  3. Админ переходит в ЛС, бот запрашивает API-токен YouGile
  4. Админ вводит токен в ЛС (токен не светится в группе)
  5. Бот получает список досок:
      0 досок → сообщение «создайте доску в YouGile»
      1 доска → автоматически привязывает её
      >1 досок → inline-клавиатура для выбора
  6. Канбан настроен

Способ Б — пошагово в группе:
  /setup_kanban → email → пароль (пароль удаляется из чата после ввода)

Способ В — прямой ввод токена:
  /kanban_token ТОКЕН ID_ДОСКИ


3. Привязка пользователей YouGile
-----------------------------------
Чтобы бот мог назначать задачи на конкретных участников, каждый участник
выполняет:

  /link_yougile

Бот показывает список пользователей YouGile. Участник выбирает себя —
его Telegram ID связывается с YouGile User ID.


4. Ежедневная работа: умный ассистент в чате
-----------------------------------------------
Участники просто пишут сообщения в чат. Бот автоматически:

Анализирует тональность и риски
  - Каждое сообщение проходит через LLM (OpenAI / Gemini / GigaChat)
  - Определяется настроение отправителя
  - Если detected риск (конфликт, выгорание, срыв сроков) — бот постит
    предупреждение в чат

Обрабатывает естественно-языковые запросы к канбан-доске

  Участник может написать, например:
    «Создай задачу: подготовить отчёт, исполнитель Иван»
      → бот создаёт карточку в YouGile

    «Покажи мои задачи»
      → бот выводит список карточек, назначенных на участника

    «Перенеси дедлайн задачи Х на пятницу»
      → бот меняет deadline

    «Закрой задачу Y»
      → карточка перемещается в «Готово»

    «Добавь комментарий к задаче Z»
      → комментарий появляется в YouGile

Система согласования:
  Если участник (не админ) пытается создать задачу на другого человека —
  бот запрашивает подтверждение у админа через reply-механизм:
  «Ответьте "да" для подтверждения заявки #123»


5. Пульс-опросы (Activity / Pulse Polls)
-------------------------------------------
Встроенная система ежедневных анонимных опросов настроения.

Включение:
  /activities_on

Настройка времени (МСК):
  /pulse_time 17:00

Настройка длительности:
  /pulse_close_after 60    (автозакрытие через 60 минут)

В назначенное время бот задаёт вопрос (например, «Какая сейчас погода
в твоём проекте?») с вариантами от 1 до 5. Ответы полностью анонимны —
Telegram ID хэшируется. После закрытия публикуется сводка со средним
баллом и распределением.

Ручной запуск:
  /pulse                — для демонстрации

Принудительное закрытие:
  /pulse_results


6. Просмотр рисков
-------------------
  /risks

Бот показывает 10 последних рисков, обнаруженных анализом тональности.


7. Сводная таблица команд
---------------------------
Команда                      Роль          Что делает
─────────────────────────────────────────────────────────────────────
/i_am_director                Любой         Создать команду
/setup_yougile                Админ         Настроить YouGile через ЛС
                                            (рекомендуется)
/setup_kanban                 Админ         Настроить YouGile пошагово
                                            в группе
/kanban_token TOKEN [BOARD]   Админ         Прямой ввод токена
/kanban_board [BOARD_ID]      Админ/любой   Сменить или посмотреть
                                            доску
/kanban_status                Любой         Статус подключения
/link_yougile                 Любой         Привязать аккаунт YouGile
/risks                        Любой         Показать риски
/activities_on                Участник      Включить пульс-опросы
/activities_off               Участник      Выключить пульс-опросы
/pulse_time HH:MM             Участник      Время опроса
/pulse_close_after N          Участник      Минут до автозакрытия
/pulse                        Участник      Запустить опрос сейчас
/pulse_results                Участник      Закрыть опрос и показать
                                            итоги


8. Архитектурно
-----------------
- Единый бот (один токен Telegram) обрабатывает и групповые, и личные
  сообщения
- Разделение через фильтры: групповые хэндлеры помечены GroupOnly() и
  срабатывают только в group / supergroup
- LLM-провайдер определяется per-user; если у пользователя нет своего
  ключа — используется ключ владельца
- Все состояния FSM живут в памяти (MemoryStorage) — при перезапуске
  FSM сбрасываются, но данные в БД сохраняются
- YouGile API используется для всех операций с канбан-доской (создание
  карточек, перемещение, дедлайны, комментарии, назначения)
